Recevez des documents et annonces

Votre gestionnaire immobilier peut vous envoyer des documents et annonces qui vous concernent. Un document peut par exemple contenir le réglement d'immeuble ou le contrat de location. Une annonce a une période validité et peut par exemple se référer à une réparation dans l'immeuble ou dans votre appartement. Même en vacances, vous avez l'assurance d'être au courant des évènements concernant votre immeuble et votre domicile.

Réservez votre buanderie commune

Réservez vos heures dans la buanderie commune de votre immeuble. Affichez vos réservations pour les prochains jours. Equa Phi assure le respect des quotas par ménage. Les cases horaires qui sont libres restent cependant réservables jusqu'à 24h à l'avance, même en cas de dépassement de quota.

Envoyez des bons de travail

Envoyez des bons de travail à votre gestionnaire immobilier. Suivez l'état de votre bon de travail. Si votre gestionnaire immobilier a besoin de plus d'informations sur votre demande, il/elle peut éventuellement engager une conversation par e-mail avec vous.

Emménagez en scannant un poster

Si votre gérant a affiché un poster dans votre immeuble vous invitant à utiliser Equa Phi, vous pouvez suivre les instructions affichées.

Emménagez en scannant votre QR-code personnalisé

Si votre gérant vous a donné accès à votre QR-code personnalisé, alors vous pouvez simplement: 1️⃣ Créer un compte dans Equa Phi, 2️⃣ scanner votre QR-code avec l'appli Equa Phi, et 3️⃣ optionnellement entrer le code secret qui vous a été attribué.

Comment scanner votre QR code

Vous avez plusieurs options: (1) Si votre agent ou votre gestionnaire immobilier vous présente votre code QR sur son appareil mobile, scannez simplement cet écran à l'aide d'Equa Phi. (2) Si vous avez reçu votre code QR sous forme papier, scannez-le à l'aide de votre application. (3) Si le code QR est sous forme numérique, si vous avez un deuxième appareil mobile ou ordinateur, scannez ce deuxième écran. Sinon, vous pouvez photocopier l'écran de votre appareil mobile puis scanner la photocopie, ou envoyer votre code QR à un co-résident et scanner son écran.

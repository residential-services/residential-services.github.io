01. Bewirtschaftung

Immobilienbestand verwalten

Fügen Sie Ihrem Inventar verwaltete Gebäude hinzu. Adressen werden über Google Maps überprüft. Anmerkungen an Gebäude anhängen. Die Anwesenheit von Annehmlichkeiten in Gebäuden angeben, z. B. gemeinsame Wäschereien. Die Angabe der Anwesenheit von Wäschemöglichkeiten in Ihren Gebäuden aktiviert automatisch die Buchungsfunktion für Ihre Bewohner. Stellen Sie für jede Wäschereieinrichtung die Zahl der Waschmaschinen ein und wählen Sie einen Buchungskalender aus vordefinierten Vorlagen.

02. Bewirtschaftung

Die Aufenthalte der Bewohner verwalten

Fügen Sie einen Aufenthalt einem Gebäude hinzu, indem Sie die Einheitskennung angeben, und auch das Einzugsdatum und den Namen des Bewohners, der den Miet- oder Verkaufsvertrag unterzeichnet hat. Fügen Sie den einzelnen Aufenthalten Notizen hinzu. Beenden Sie einen Aufenthalt, indem Sie das Auszugsdatum einstellen. Geben Sie optional einen Mietbetrag für die automatisierte Mietsammlung an.

03. Bewirtschaftung

Miete automatisch einziehen

Sammeln Sie Miete mit minimalem Aufwand, indem Sie einfach Ihre Bankkontonummer und die Mietbeträge eingeben. Verwalten Sie Mietzahlungen in Ihrem Dashboard auf Stripe.com. Ermöglichen Sie den Haushaltsmitgliedern, die Miete aufzuteilen, und ihren Anteil an der Miete zu ändern, wann sie möchten. Ermöglichen Sie den Einwohnern, ihr Quellbankkonto zu ändern, ohne Sie hineinzuziehen. Erlauben Sie den Einwohnern, zu Equa Phi-Zahlungen zu wechseln, auch wenn sie andere Zahlungsmethoden eingerichtet haben. Ändern Sie das Bankkonto, das Sie zum Sammeln von Mieten verwenden, ohne Ihre Bewohner benachrichtigen zu müssen. Die meisten Länder der Eurozone werden durch SEPA-Lastschriftzahlungen unterstützt.

04. Bewirtschaftung

Integrieren Sie Equa World mit anderen Buchhaltungs- und Immobiliensoftware

Wenn Sie die Mietersammlungsfunktion verwenden, integrieren Sie Equa World mit Ihrer Buchhaltungssoftware durch die Berichtsmöglichkeiten von Stripe.com. Wenn sie andere Immobiliensoftware benutzen, senden Sie an Equa World Ihre vorhandenen Immobiliendaten über Gebäude und Aufenthalte, indem Sie CSV-Dateien in die Equa Delta-App hochladen. Erstellen Sie diese CSV-Dateien, indem Sie aus Quellen, z. B. Microsoft Excel, Apple Numbers, Google Sheets, Airtable, oder durch programmgesteuerte Abfragen der Datenbanken anderer Immobiliensoftware exportieren.

05. Bewirtschaftung

Organisieren Sie Ihr Team

Jedes Mitglied des Property Management-Teams kann auf eine oder mehrere verschiedene Rollen für eine effiziente Arbeitsteilung spezialisiert werden. Die Person, die behauptet, dass das Property Management Company in Equa World die "Leiter"-Rolle zugewiesen wird. Leiter haben vollen Zugang zu allen Equa World-Funktionen, und sie können andere Personen an das Team hinzufügen, sei es in einer "Leiter"-Rolle oder mindestens einem von "Agenten"-, "Verwalter der Bewohnerbeziehungen"-, "Bestandsverwalter"-, "Editor"- oder "Personalleiter"-Rollen. Die vollständige Erläuterung jeder Rolle ist in der Equa Delta-App bereitgestellt.

06. Bewohnererlebnis

Gemeinsame Wäschereien verwalten

Aktivieren Sie die Zeitfenster-Reservierungsfunktion für Ihre Bewohner, indem Sie vorhandene gemeinsame Wäschereien angeben. Geben Sie für jede Einrichtung die Anzahl der reservierbaren Maschinen an und wählen Sie einen Reservierungskalender aus vordefinierten Vorlagen aus. Wenn es mehrere Maschinen gibt, kennzeichnen Sie sie physisch von "1" aufwärts.

07. Bewohnererlebnis

Senden Sie Dokumente und Ankündigungen an die Bewohner

Mit die Equa-Delta-Anwendung können Sie Dokumente und Ankündigungen an die Bewohner senden, die in einer bestimmten Wohneinheit, einem Gebäude, einer Gruppe von Gebäuden oder allen Ihren Gebäuden sich aufhalten. Ein Dokument kann beispielsweise ein Mietvertrag oder ein Hausordnungsdokument sein. Ankündigungen haben ein Verfallsdatum, und können sich auf ein zeitbegrenztes Ereignis wie beispielsweise eine Reparatur oder eine lokale Versammlung beziehen. Die Bewohner können ihre Dokumente und Ankündigungen mit der Equa Phi-App lesen.

08. Bewohnererlebnis

Erhalten Sie Arbeitsaufträge von den Bewohnern

Erhalten Sie Arbeitsaufträge von Ihren Bewohnern, und verfolgen Sie deren Fortschritt, indem Sie den Arbeitsauftragsstatus einstellen und einen Textanmerkung anhängen. Um nach weiteren Informationen zu einem Auftrag zu fordern, kopieren Sie einfach die E-Mail-Adresse des Absenders und senden Sie ein E-Mail.

09. Bewohnererlebnis

Bestehende Bewohner effizient einbinden

Wir empfehlen, dass Sie unsere Poster-basierte Einbindungsmethode für Ihre bestehenden Einwohner verwenden. Legen Sie einfach Poster in Ihren Gebäuden auf und lassen Sie Bewohner den QR-Code scannen, der das Poster enthält. Senden Sie eine E-Mail an support@equa.world, damit wir für jedes Ihrer Gebäude maßgeschneiderte Poster erstellen können, die Sie dann ausdrucken und in verwalteten Gebäuden aufstellen werden. Beachten Sie, dass Sie diese Einbindungsmethode auch für neue Bewohner verwenden können; in diesem Fall müssen Bewohner auf ihr Einzug warten.

10. Bewohnererlebnis

Neue Bewohner einbinden

Wir empfehlen, dass Sie die Poster-basierte Einbindungsmethode verwenden, aber falls Sie einen Bewohner vor dem Einziehen einbinden möchten, haben Sie zwei Möglichkeiten: (1) Anwohner ihren QR-Code auf Ihrem Handy mit der Equa Delta-App anzeigen; dies ist praktisch, wenn sie dem Bewohner persönlich begegnen, zum Beispiel bei der Unterzeichnung des Vertrags. (2) Den Bewohnern ihren QR-Code in digitaler oder Papierform senden, indem Sie den QR-Code zuerst in einer Datei speichern. Für beide Optionen können Sie einen geheimen Text auswählen, den Sie dem Bewohner mitteilen, um die Einbindungsvorgang sicherer zu machen.

Die Bewohner-Erlebnis-App

Die Bewohner haben ihre eigene App, die sie KOSTENLOS in ihrem täglichen Leben nutzen können. Die Standard-App, die wir Bewohnern anbieten, ist Equa Phi. Darüber hinaus können Sie Bewohnern, die von einer oder mehreren Ihrer Filialen betreut werden, Ihre eigene Marken-App anbieten. Marken-Apps kosten 82 € pro Monat; bitte kontaktieren Sie uns unter support@equa.world.
